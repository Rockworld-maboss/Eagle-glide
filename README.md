# Academic CV Updater (Python)

## 📘 Description
This project is a Python-based tool that allows users to easily update their academic accomplishments—such as certificates, diplomas, and degrees—directly into a structured CV file.  
The CV data is stored in JSON format and is automatically updated when new qualifications are added.

This project demonstrates Python skills in file handling, data structures, and automation.

---

## ✨ Features
- Add new academic qualifications (certificates, diplomas, degrees)
- Automatically updates CV data
- Uses structured JSON storage
- Simple command-line interface
- Easy to extend (PDF, Word, web app, etc.)
